<--Описание-->
Веб-приложение AskAndLearn, которое является площадкой для публикации вопросов и получения на них ответов.
Учебный проект для Яндекс Лицея.

<--Зависимости-->
Программа работает на языке Python (3.12 и выше)
Все необходимые библиотеки находятся в файле requirements.txt

<--Инструкция по установке-->
На Windows:
  1. Клонируйте проект с GitHub по ссылке: https://github.com/Yarbick/AskAndLearn.git
  2. Установите компилятор Python
  3. Зайдите в cmd в папке с проектом и пропишите команду (установка зависимостей):
      pip install -r requirements.txt
  4. Создайте файл .env в корневой папке проекта и запишите туда:
      SECRET_KEY="<секретный ключ (набор любых символов)>"

Теперь у вас есть готовое приложение. Попробуйте запустить его через localhost
